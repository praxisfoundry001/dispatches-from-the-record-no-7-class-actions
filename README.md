# Dispatches from the Record: Clerk Notes No. 7 — Class Actions

## What is this?

This is the seventh note in the *Dispatches from the Record: Clerk Notes* series. It offers a plain-language structural reading of class actions.

## Why should I care?

Class actions matter because some harms are hard to see when every person has to proceed alone. This note explains why courts sometimes need a procedural way to perceive many related claims together without pretending the people behind those claims are literally one person.

## What is the claim?

The central claim is that class actions operate as controlled perceptual compression. They let a court see related claims at scale while preserving individual identity downstream. The hard boundary is between collective perception and individual erasure.

## What is this not?

This is not legal advice, a replacement for Rule 23, a civil-procedure treatise, or a case-specific certification guide. It does not decide whether any class should be certified. It offers a structural interpretation of what class-action doctrine is doing.

## How was this done?

The note uses the Constitutional Physiology / Structural Flow lens to read class actions as a procedural form of pattern recognition: many harm signals become visible together only if the system can aggregate them without dissolving the individuals who generated them.
